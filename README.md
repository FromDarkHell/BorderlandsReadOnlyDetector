# BorderlandsReadOnlyDetector

# About
A tool for Borderlands 2 (and TPS, untested) to notify the user of their read-only state if the save is probably unintended to be in read-only. 

# Requirements
- [PluginLoader](https://github.com/c0dycode/BorderlandsPluginLoader)
- [CommandInjector](https://github.com/c0dycode/BL-CommandInjector)
